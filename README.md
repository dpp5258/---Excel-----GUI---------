# 日常任务提醒系统

基于 Excel 存储的 GUI 桌面任务管理工具，使用 PyQt6 + openpyxl 构建。

## 快速开始

```bash
pip install -r requirements.txt
python gui.py
```

数据文件 `任务清单.xlsx` 首次运行时自动创建。

## 功能概览

| 功能 | 说明 |
|------|------|
| 新增 / 编辑 / 删除任务 | 弹窗填写，标题必填，截止时间支持日历选择器 |
| 三表管理 | 待办任务 → 已完成任务 → 归档任务，数据永不丢失 |
| 优先级排序 | 优先级降序 → 截止时间升序 → ID 升序 |
| 截止时间颜色标识 | ID 后标识格按紧迫度显示 蓝 / 绿 / 橙 / 红 |
| 剩余时间显示 | 表格列 + 弹窗实时计算，精确到小时 |
| 关键字搜索 | 标题模糊匹配，实时过滤 |
| 自动备份 | 每次写入前自动备份，保留最近 10 份 |
| 日志记录 | 操作日志写入 `control.log` |
| 数据修复 | 表头损坏时自动重建并恢复有效数据行 |

## 颜色标识规则

| 剩余时间 | 颜色 |
|----------|------|
| > 14 天 | 蓝色 |
| 7 ~ 14 天 | 绿色 |
| 3 ~ 7 天 | 无填充 |
| < 3 天 | 橙色 |
| 已过期 | 深红 |
| 无截止时间 | 无填充 |

## 项目结构

```
├── model.py       数据模型 — Task 实体、截止状态枚举、剩余时间计算、日志工具
├── storage.py     Excel 存储层 — 三 Sheet 读写、自动备份、表头校验与修复
├── controller.py  业务控制层 — CRUD、状态流转（完成/撤回/归档）、搜索排序
├── gui.py         GUI 界面层 — PyQt6 主窗口、新增/编辑弹窗、完成确认弹窗
├── requirements.txt
└── README.md
```

## Excel 数据结构

三张 Sheet 共用 9 列：

`ID | 任务标题 | 任务内容 | 创建时间 | 截止时间 | 优先级 | 备注 | 完成状态 | 完成情况`

| Sheet | 用途 |
|-------|------|
| 待办任务 | 当前未完成的任务 |
| 已完成任务 | 已完成但未归档的任务 |
| 归档任务 | 已归档的历史记录 |

## 运行环境

- Python 3.10+
- Windows / macOS / Linux
